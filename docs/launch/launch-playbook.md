# Launch Playbook

## Launch-day order

1. Publish GitHub release `v0.1.0` with release notes and demo GIF.
2. Publish PyPI `0.1.0` package.
3. Post launch threads:
   - X / LinkedIn
   - Reddit (`r/PowerSystems`, `r/MachineLearning` if content fits rules)
   - Chinese communities (Juejin / Zhihu / V2EX)

## Unified CTA

`Star + Try + Share a case`

## Post templates

- Hook: "Natural-language power-system analysis on top of pandapower."
- Demo: attach `docs/assets/quick-demo.gif`.
- Value bullets:
  - AC/DC/3-phase, short-circuit, topology, contingency
  - Scenario save/load/compare/undo
  - JSON + PNG export for reproducible reports
- CTA: Star repo + run quickstart + share your network case.
