## [0.1.0] - 2026-02-28

### Added

- Initial public open-source release.
- CLI for natural-language power system analysis workflows.
- Built-in network discovery/switching and scenario state management.
- AC/DC/three-phase power flow, short-circuit, topology, contingency tools.
- Plotting, export, and doctor checks.
- Governance docs, CI workflow, release workflow, and contribution templates.
